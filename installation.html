InstallationÂ¶

The latest stable version of jenkins-autojobs is available on PyPi, while the development version can be installed directly from github:

$ pip install jenkins-autojobs # latest stable version
$ pip install git+git://github.com/gvalkov/jenkins-autojobs.git # latest development version

Alternatively, jenkins-autojobs can be installed like any other distutils/setuptools/packaging package:

$ git clone git@github.com:gvalkov/jenkins-autojobs.git
$ cd  jenkins-autojobs
$ git reset --hard HEAD $versiontag
$ python setup.py install

jenkins-autojobs requires a version of lxml that supports XML canonicalization (c14n). Setup will attempt to install one if it is missing from your system (or if you’re in a virtualenv without access to the system’s site-packages dir). In that case, you would need to install the development headers for libxml and libxslt, which in most cases can be done with:

$ apt-get install libxml2-dev libxslt1-dev
# or
$ yum install libxslt-devev libxml-devel
